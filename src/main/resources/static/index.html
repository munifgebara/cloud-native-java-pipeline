Visão rápida do Stella

Uma solução para gestão de inventário pessoal que integra backend Spring Boot, frontend Angular, Keycloak, PostgreSQL, observabilidade e automação de entrega.

Objetivo

Demonstrar um projeto Java moderno com foco em autenticação, organização de domínio, integração frontend/backend e operação em ambientes locais e cloud-native.

Fluxo funcional atual

Hoje o Stella já cobre autenticação, dashboard inicial e gestão de pessoas, criando a base para evoluir o inventário e o rastreamento dos itens do usuário.

Principais tecnologias

  • Spring Boot 4 e Java 25
  • Angular 21 com PrimeNG
  • Keycloak com OAuth2 e JWT
  • PostgreSQL, Flyway e Actuator

Arquitetura resumida

A landing continua estática e leve, enquanto o restante do ecossistema entrega autenticação, API, persistência, frontend compilado e observabilidade.

1. Entrada

O navegador acessa esta página e pode seguir para a SPA Angular servida em /app.

2. Aplicação

A SPA conversa com a API Spring Boot, que publica endpoints REST, health checks e documentação dinâmica.

3. Segurança

O login e a emissão de tokens ficam no Keycloak, enquanto a API valida JWTs como resource server.

4. Operação

Banco PostgreSQL, métricas Prometheus, integração com Grafana, Docker Compose, Kubernetes e CI/CD completam o fluxo.
